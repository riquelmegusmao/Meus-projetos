# Botão Neon com Animação 

Um botão com efeito neon verde no hover, feito com HTML e CSS puro.

##  Funcionalidades
- Hover com `transition` e `transform`
- Efeito neon com `box-shadow`
- Texto com `::before`

##  Tecnologias
- HTML5
- CSS3
